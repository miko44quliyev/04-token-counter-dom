How an LLM Works

From the moment you press Enter to the first word of the response, here is what happens inside a large language model.

  1. 1

    User Input

    You type a prompt and submit it. This raw text is sent to the model's API as plain text waiting to be interpreted.

  2. 2

    Tokenisation

    The tokeniser breaks the text into tokens: small chunks of roughly 3 to 4 characters.

    ["Hello", ",", " world", "!"]

    Each token is mapped to a unique integer index value.

  3. 3

    Embedding

    Each token ID is looked up in an embedding table and converted into a high-dimensional vector list. These numbers encode the semantic meaning of the token in a form the neural network layers understand.
